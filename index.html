Campus Safety App - Project Overview

The Campus Safety App was developed to enhance student life at Lawrence University by streamlining request management. The existing system relied on phone calls, leading to inefficiencies and confusion. This mobile app modernizes the process with a structured, digital approach.

Project Features

  • 🚀 Two Mobile Apps (React Native): One for students to submit requests, another for officers to manage them.
  • 🌎 Real-Time Location & Maps: Integrated with Google APIs for location tracking and address searches.
  • 🔐 Secure Authentication: Uses bcrypt password hashing and automated email verification.
  • 📊 Admin Dashboard: Web-based management system mirroring the database for easy monitoring.
  • ⚡ Scalable AWS Deployment: Backend runs on EC2, behind an Application Load Balancer with Auto Scaling.
  • 📂 MySQL Database (AWS RDS): Centralized storage for requests, users, and logs.

Tech Stack

  • Frontend: React Native
  • Backend: Java Spring Boot
  • Database: MySQL (AWS RDS)
  • Cloud Hosting: AWS EC2 + ALB + Auto Scaling
  • Security: bcrypt password hashing, JWT authentication
  • Maps & Location: Google APIs (Geocoding, Maps, Places)

Project Impact

This project significantly improved efficiency for both students and Campus Safety officers. By providing a modern, real-time solution, it reduced response times and streamlined operations. The experience of designing and implementing a full-stack system strengthened my skills in scalable architecture, security, and user-centric software development.
